# ai-instructions

A collection of skills, agents, and instructions for Claude Code and GitHub Copilot.

## Contents

| Folder | Description |
|--------|-------------|
| `skills/` | Claude Code skills — features activated via slash commands |
| `agents/` | Specialized AI agents — `.agent.md` files for orchestration |
| `instructions/` | System instructions and reusable prompts |

## Available skills

### `codebase-expert-generator`

Analyzes a project and generates a `codebase-expert.agent.md` file documenting the architecture, tech stack, conventions, and workflows. This file can then be loaded as context by other AI agents.
